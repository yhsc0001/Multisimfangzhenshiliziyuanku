## Multisim简介

Multisim是美国国家仪器（NI）有限公司推出的一款基于Windows的仿真工具，主要用于板级模拟/数字电路板的设计工作。它支持电路原理图的图形输入和电路硬件描述语言输入方式，并提供了丰富的仿真分析能力，帮助用户在设计过程中进行电路验证和优化。
